# Power BI Desktop Version – Domino’s Analytics Project

This folder contains the Power BI Desktop implementation of the Domino’s Analytics Project using the Kaggle `pizza_sales.csv` dataset.

---

## 📌 Files Included

### 1. `Dominos_Analytics.pbix`
The main Power BI report file.

### 2. `dax_measures.md`
All DAX measures used in the report.

### 3. `powerbi_report_pages.md`
Documentation of each report page and its visuals.

### 4. `dataflow_structure.md`
Details of the data model, transformations, and relationships.

### 5. `report_screenshots/`
Screenshots of the Power BI report pages.
